Tipos de software: software propietario y software libre

Software propietario:

Es aquel que es desarrollado y distribuido por empresas que mantienen el control sobre su código fuente. Los usuarios deben aceptar licencias específicas que restringen la modificación, redistribución o copia del software. Ejemplos comunes incluyen Microsoft Windows, macOS y Microsoft Office. Las principales características del software propietario son: Mayor soporte técnico profesional. Compatibilidad con sistemas y hardware comerciales. Innovación respaldada por recursos significativos.

Software libre:

Se distribuye con licencias que permiten a los usuarios modificar, compartir y usar el software sin restricciones. Ejemplos populares incluyen el sistema operativo Linux y programas como GIMP y LibreOffice. Sus características incluyen: Transparencia y seguridad por acceso al código fuente. Costo bajo o gratuito. Flexibilidad para adaptarse a necesidades específicas​

Tipos de software: sistema operativo, software de programación y programas de aplicación

Sistemas operativos:

Son la base del funcionamiento de cualquier computadora, gestionando hardware y software. Ejemplos incluyen Windows, Linux y macOS. Sus características clave son: Gestión de recursos del sistema (procesador, memoria, dispositivos). Interfaces de usuario gráficas o de línea de comandos. Seguridad y multitarea.

Software de programación:

Facilita la creación de nuevos programas. Incluye herramientas como editores de texto, compiladores e IDEs (Entornos de Desarrollo Integrados). Ejemplo: Visual Studio Code (editor de código) y GCC (compilador).

Programas de aplicación:

Diseñados para tareas específicas, como procesadores de texto (Microsoft Word), hojas de cálculo (Excel) o navegadores web (Google Chrome). Están orientados al usuario final y suelen estar optimizados para un sistema operativo específico

Funciones principales de Windows

El sistema operativo Windows destaca por: Gestión de archivos y almacenamiento: Proporciona herramientas como el Explorador de Archivos para organizar datos en discos locales y en la nube (OneDrive). Interfaz gráfica amigable: Ofrece accesos rápidos mediante menús y ventanas, facilitando la interacción incluso a usuarios sin experiencia técnica. Compatibilidad amplia: Soporta una gran variedad de aplicaciones y dispositivos de hardware. Seguridad: Incluye Windows Defender y actualizaciones frecuentes para proteger contra amenazas cibernéticas. Facilidades para redes: Simplifica la conexión a redes domésticas y empresariales con opciones como el Centro de Redes y Recursos Compartidos​
