Hardware y sus características

El hardware comprende todos los componentes físicos de una computadora, los cuales se clasifican en función de su funcionalidad: Hardware básico: Incluye los elementos esenciales para el funcionamiento de la computadora, como el procesador, memoria RAM, disco duro y tarjeta madre. Hardware complementario: Periféricos y dispositivos que amplían las capacidades del equipo, como impresoras, cámaras web o altavoces. Clasificación funcional: Entrada: Teclado, mouse, escáner. Salida: Monitor, impresora, bocinas. Procesamiento: CPU, GPU. Almacenamiento: Discos duros (HDD o SSD), USB. Híbridos: Pantallas táctiles, unidades de disco

Componentes principales de una computadora

Tarjeta madre (Motherboard): Es el núcleo donde se conectan todos los componentes. Contiene ranuras para CPU, RAM, tarjetas de expansión y almacenamiento. Permite la comunicación entre los elementos internos del sistema Unidad Central de Procesamiento (CPU): Conocido como el "cerebro" del sistema, realiza operaciones lógicas y de control. Sus características incluyen velocidad (GHz), cantidad de núcleos y eficiencia energética
Memoria RAM: Es volátil y almacena datos temporalmente para acelerar la ejecución de programas. Las capacidades modernas oscilan entre 8 GB y 64 GB
Memoria ROM: No volátil, contiene instrucciones básicas del sistema como el firmware. Es esencial para el arranque del equipo【8】. Discos duros (HDD y SSD):
HDD: Mayor capacidad por menor costo, pero más lento.
SSD: Más rápido y resistente, aunque más caro.
Periféricos:
Entrada: Teclado, mouse, micrófonos. Salida: Monitor, impresoras. Entrada/Salida: Pantallas táctiles, dispositivos USB
Otros componentes: Tarjeta de video (GPU): Optimiza el procesamiento de gráficos y tareas intensivas. Unidades ópticas (CD/DVD/Blu-ray): Cada vez menos comunes, utilizadas para discos físicos Gabinete: Aloja todos los componentes y facilita el flujo de aire
